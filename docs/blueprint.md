# **App Name**: Data Sift

## Core Features:

- Data Entry: Allow users to add new data records with relevant fields.
- Advanced Search: Implement a search bar for quickly finding entries based on multiple criteria.
- Results Display: Display search results in an easily navigable, clear list format.
- Record Details: Provide a detailed view for each record with all available information.

## Style Guidelines:

- Background color: Dark blue (#1A237E), providing a professional and modern feel.
- Primary color: Light gray (#CFD8DC), for content and interface elements to ensure readability and contrast on the dark background.
- Accent color: A slightly brighter shade of gray (#90A4AE), used for interactive elements to guide the user without distracting from the data.
- Font pairing: 'Inter' for headlines, a sans-serif, and 'PT Sans' for body, a humanist sans-serif that provides warmth and readability.
- Use simple, clear icons for navigation and actions, ensuring they are easily understandable.
- Maintain a clean and efficient layout that supports data visibility and easy interaction, prioritizing function over excessive design elements.
- Subtle transitions for search and data loading to improve the user experience without being distracting.
